# Tuščios būsenos (UX šablonas)

Vienodas šablonas pagrindiniams view:

1. **Ikona** (lucide, `text-slate-300` / `text-slate-400`).
2. **Vienas sakinys** — kas trūksta (pvz. „Dar nėra užsakymų“).
3. **Vienas CTA** — pirminis veiksmas (pvz. „Naujas užsakymas“), jei tinka kontekstui.

Taikyti bent: **Užsakymai**, **Klientai**, **Apžvalga** (kai nėra duomenų grafikams). Esami komponentai gali būti palaipsniui suvienodinti be elgsenos keitimo.
